# 📰 News App

A simple and responsive News Application built using **HTML, CSS, and JavaScript** that fetches real-time news using a News API.

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* News API
 
---
 
## 🚀 Features

* 🔍 Search news by keyword
* 🌍 Fetch latest headlines
* 📱 Fully responsive design
* ⚡ Fast and lightweight

---

## ⚠️ Important Note

> This project uses a **free News API**, which only works on `localhost`.

Due to API restrictions, the app **cannot be deployed on live servers** (like Netlify, Vercel, etc.).

---

## 💡 How to Run Locally

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/news-app.git
   ```

2. Open the project folder

3. Run using Live Server
   (VS Code Extension recommended)
